Welcome to GeckoLinux

GeckoLinux is a Linux spin based on the openSUSE distribution, with a focus on polish and out-of-the-box usability on the desktop.

Motivation Behind GeckoLinux

We all know that there's a profusion of Linux distributions and spins out there. So why create another one?

  1. Linux spins based on openSUSE are rarely created and seldom discovered.
  2. openSUSE is a fundamentally excellent Linux distribution, but it suffers from a suboptimal default configuration for desktop usage.
  3. The Suse Studio makes it relatively easy to create custom spins of openSUSE.

GeckoLinux Features

  • The latest stable release of the Cinnamon desktop environment with a pleasant default configuration.
  • Carefully selected open source desktop programs and proprietary media codecs pre-installed and ready to use.
  • Beautiful open source font rendering configured out of the box.
  • Much more open source and patent encumbered software available in the pre-configured repositories.

Who Are You?

I've been a user of Linux on the desktop for well over a decade. I have used many versions of openSUSE since its inception, and I used SuSE Linux before that. I've always appreciate openSUSE's flexibility and innovative infrastructure, but the profusion of "paper cuts" in the default configuration always made me think twice before recommending it to others. This projects sets out to change that.
